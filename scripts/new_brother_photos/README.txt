Replace these 5 brother photos
===============================

Copy your new PNG images into this folder and name them exactly:

  asmitha_chunchu.png
  ananya_iyengar.png
  kyle_chen.png
  lohit_potnuru.png
  lauren_nguyen.png

Then from the project root run:

  node scripts/replace-brother-photos.js

That will convert them to JPG and update public/images/brothers/current_data/.

Source files (from Cursor assets) — copy and rename:
  asmitha_chunchu-*.png  → asmitha_chunchu.png
  ananya_iyengar-*.png   → ananya_iyengar.png
  kyle_chen-*.png        → kyle_chen.png
